Creating Elements with jQuery

Creating Elements

It is best practice in jQuery to create an element(an HTML node) before adding the element to the DOM so that the element can be manipulated without having to redraw and repaint the DOM.

This is accomplished by calling jQuery and passing in a string of the tag you would like to create as the argument. For example:

1
$myDiv = $( "<div/>" );

This will create <div></div> without inserting it into the DOM. Until we add the element to the DOM, it exists only as a JavaScript object.

Note: prepending the name of a variable with $ is a stylistic choice that helps with quickly distinguishing variables that point to jQuery elements

Modifying Elements with jQuery

We create elements before adding them to the DOM so that we can manipulate them without causing the DOM to paint, draw, etc(all the things that you see happening when something in the DOM changes).

For example, we might want to add a class to an element or insert content before we add it to the DOM:

1
2
$myDiv.addClass( "myClass" );
$myDiv.html( "Wooooooo!" );

Adding it to the DOM

jQuery has several methods for adding elements to the DOM.

.html() overwrites the contents of the element it is called on, so $( "body" ).html( $myDiv ); will empty body and replace its contents with your element.

.append() will add your element after all of the other nodes in the element it is called on. $( "body" ).append( $myDiv ); will add your element after all of the elements and text nodes currently in body.

.appendTo() is similar to .append(), but it is called on the element you want to append and the element you want to append it to is passed as the argument: $myDiv.appendTo( "body" );

.after() finds an element in the DOM and adds the element passed to it after that element. Given $("header").after(<footer>); results in <header></header><footer></footer>.

Other DOM manipulation methods in jQuery include .prepend(), .prependTo(), and .before().

Removing elements from the DOM

.remove() removes the element it is called on from the DOM.

$( "body" ).remove(); will remove body from the DOM entirely.

If you've assigned an element to a variable, like we've done with $myDiv, you can call .remove() on that variable like $myDiv.remove() because jQuery is still aware of that element.
